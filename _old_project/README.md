# python-project

Thema: Aufgabenlisten-Manager (To-Do-List Manager)

Wir entwickeln einen Aufgabenlisten-Manager, mit dem sich Aufgaben einfach speichern, verwalten und filtern lassen. Nutzerinnen und Nutzer können Aufgaben hinzufügen, bearbeiten, löschen und als erledigt markieren. Zudem können Aufgaben nach Status, Priorität oder Fälligkeitsdatum gefiltert und sortiert werden.

Die Daten werden dauerhaft in einer Datei gespeichert, sodass sie auch nach einem Neustart erhalten bleiben. Ergänzend bietet das Programm Prioritätenstufen (niedrig, mittel, hoch), Fälligkeitsanzeigen sowie ein Beschreibungsfeld für jede Aufgabe.

Ziel:
Ein funktionsfähiges Python-Programm, das alle Kernfunktionen stabil ausführt und beim Start bereits drei Beispielaufgaben anzeigt.
